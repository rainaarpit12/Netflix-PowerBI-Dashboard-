# 🎬 Netflix Data Analysis Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive data visualization dashboard built using Microsoft Power BI to analyze Netflix content trends. It provides insights into movies and TV shows based on country, genre, release year, and ratings.

The dashboard includes advanced analytics features such as forecasting and decomposition tree for deeper data exploration.

---

## 🚀 Features

* 📌 KPI Cards (Total Shows, Movies, TV Shows)
* 📈 Forecasting of content trends over time
* 🌳 Decomposition Tree for root cause analysis
* 🗺️ Map visualization for geographic distribution
* 📊 Column & Pie charts for category comparison
* 🎛️ Interactive slicers for filtering (Year, Category, etc.)

---

## 📂 Dataset

The dataset contains information about Netflix titles, including:

* Title
* Type (Movie / TV Show)
* Country
* Genre (listed_in)
* Release Year
* Rating

---

## 🛠️ Tools & Technologies

* Microsoft Power BI Desktop
* Data Cleaning & Transformation (Power Query)
* DAX (Data Analysis Expressions)

---

## 📌 Key Insights

* Movies dominate the platform compared to TV shows
* Certain countries contribute significantly more content
* Content growth trend shows variation across years
* Genre distribution highlights popular categories
